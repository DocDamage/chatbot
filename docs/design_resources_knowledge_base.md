# Design Resources Knowledge Base

This document contains a curated collection of design resources extracted from `bradtraversy/design-resources-for-developers`, `manishparui/design-resources-for-developers`, and `KKshitiz/Awesome-UI-Templates`. These resources are available for future UI/UX enhancements but are not currently integrated.

---

## 🎨 Color Tools
*Tools for generating palettes, gradients, and finding color inspiration.*

- **[Coolors](https://coolors.co)** - Fast color scheme generator.
- **[Happyhues](https://www.happyhues.co/)** - Color palette inspiration in context.
- **[Huemint](https://huemint.com/)** - AI color palette generator.
- **[Gradient Hunt](https://gradienthunt.com/)** - Free open collection of color gradients.
- **[UI Gradients](https://uigradients.com/)** - Beautiful color gradients.
- **[Web Gradients](https://webgradients.com/)** - Collection of linear gradients.
- **[Material Palette](https://www.materialpalette.com/)** - Material Design color palette generator.
- **[0to255](https://www.0to255.com/)** - Find lighter and darker colors based on any color.

## 🖼️ Illustrations & Vectors

- **[Undraw](https://undraw.co/illustrations)** - Open-source illustrations for any idea.
- **[Freepik](https://www.freepik.com)** - Vectors, photos, and PSD files.
- **[Humaaans](https://www.humaaans.com/)** - Mix-&-match illustrations of people.
- **[DrawKit](https://www.drawkit.io/)** - Beautiful, free vector illustrations.

## 💻 Template Collections & Starters
*Ready-to-use templates for rapid development.*

- **[learning-zone / website-templates](https://github.com/learning-zone/website-templates)**
  - Massive collection of 150+ HTML5/CSS3 templates.
  - Covers various categories: Landing pages, Portfolios, Agencies, Dashboards.
  - Useful for: Quick prototyping or finding layout inspiration for specific page types.

- **[Cveinnt / LiveTerm](https://github.com/Cveinnt/LiveTerm)**
  - **Terminal-style** website template built with Next.js.
  - Highly customizable via a single `config.json`.
  - Useful for: Creating a "hacker" or developer-centric portfolio/resume site (fits the 'Antigravity' theme well).

- **[arthelokyo / AstroWind](https://github.com/arthelokyo/astrowind)**
  - Astro 5 + Tailwind CSS starter.
  - Production-ready with Dark Mode support, Blog, and Landing Page widgets.

- **[YaoApp / yao](https://github.com/YaoApp/yao)**
  - **AI-Powered App Engine**: specifically tags "chatbot" and "admin-dashboard".
  - Useful for: Backend/Dashboard logic references for AI apps.

- **[puikinsh / Adminator](https://github.com/puikinsh/Adminator-admin-dashboard)**
  - Clean Admin Dashboard template (Bootstrap 5).
  - Useful for: Complex data tables and settings pages.
