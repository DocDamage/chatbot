# Creative Writing and Roleplay

The creative track supports original fiction drafting, roleplay sessions, long-form project work, revision passes, continuity tracking, branching, exports, prompt packs, and privacy controls.

## Creative Modes

- `creative_writing` is for drafting scenes, chapters, outlines, revisions, character work, worldbuilding, continuity summaries, and exports.
- `roleplay` is for turn-based scenes with a player character, assistant character, narrator mode, scene location, active cast, goals, inventory, boundaries, pause/resume, branches, and summaries.

## Mature Fiction Policy

Mature and Adult Fiction ratings are opt-in controls for release-safe fictional writing. The system does not bypass boundaries. It blocks illegal, exploitative, minor-involved, real-person sexual, non-consensual sexual, privacy-invasive, and self-harm enabling requests, then offers safer rewrites such as fade-to-black, fictional adult substitutes, or non-explicit tension/aftermath.

## Story Bible and Continuity

Story bible context can include characters, locations, factions, lore rules, timeline events, chapters, scenes, style guide notes, rating rules, and continuity notes. Pinned items are injected into creative responses so later drafts preserve canon details.

## Long-Form Workflow

Projects can move through premise, logline, outline, beat sheet, chapter outline, scene list, chapter draft, continuity review, revision pass, and final export. Draft versions are stored per scene, alternate takes are stored as branches, and exports support Markdown, plain text, and JSON with metadata.

## Prompt Packs

Prompt packs store reusable genre, trope, character archetype, scene type, and tone profile prompts. Packs can be created, applied to a request, exported, imported, and deleted separately from project drafts.

## Privacy Controls

Sensitive sessions can be marked local/private, disable analytics, set retention days, request log redaction, and delete project data. Redaction handles common email and phone patterns before logs are stored or shared.

## Degraded Mode

When only a template fallback provider is available, creative mode reports degraded behavior. Lightweight scaffolding still works, but long-context continuity, nuanced character voice, and high-quality prose need a capable model provider.

## Manual QA Checklist

- Draft regular fiction with a story bible note and verify the note appears in the response.
- Draft dark horror, epic fantasy, space opera, western, and romance scenes with genre presets.
- Enable Mature or Adult Fiction only intentionally and verify boundary redirects for unsafe requests.
- Start a roleplay scene, pause, resume, branch, summarize, and reset it.
- Create a project, add at least two scene versions, restore a branch, and export Markdown plus JSON.
- Apply a prompt pack, export/import it, and delete it.
- Mark a project private and verify analytics are disabled and deletion removes the project.
