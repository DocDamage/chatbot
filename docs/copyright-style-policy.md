# Copyright-Safe Creative Style Policy

Creative Writing mode may use broad genre, tone, structure, and atmosphere descriptors, but it must not continue protected works, reuse protected worlds or characters, or imitate a living author exactly.

Allowed transformations:
- "Lord of the Rings type" -> mythic secondary-world epic fantasy
- "Stephen King type" -> small-town supernatural psychological horror
- "Star Wars type" -> space-fantasy rebellion with mystical orders and frontier worlds

Blocked requests:
- Direct sequels, continuations, or scenes set in protected worlds.
- Requests to use protected characters, places, lore, or exact story events.
- Exact imitation of a living author's style, prose voice, or signature expression.

Safe response pattern:
1. Briefly name the limitation.
2. Offer an original substitute using generic descriptors.
3. Keep new names, settings, factions, plot events, and prose voice original.
