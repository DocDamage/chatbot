# Suno Structure Tags

Useful structure prompts include `[Intro]`, `[Verse]`, `[Pre-Chorus]`, `[Hook]`, `[Bridge]`, `[Breakdown]`, and `[Final Hook]`.

Tags should describe arrangement energy:

- intro atmosphere
- sparse verse
- hook lift
- bridge contrast
- final hook with doubled energy
